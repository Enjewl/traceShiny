How to upload data

Select the data to upload. There are two steps:

  1. Select the “Data upload type” (see table below).

  2. Select the file(s) to upload.

Data upload type

Data type Description
FSA The raw file from the Genetic Analyzer instrument. Must be .fsa file format.
EXAMPLE DATA Run an in-built example fsa file experiment, which consists a Huntington’s disease cell line model with an unstable HTT repeat (refer to our publication).
